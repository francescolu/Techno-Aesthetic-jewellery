# Techno-Aesthetic-jewellery

技术美学视角下20世纪商业珠宝的审美机制研究

## 项目说明

本博士论文研究项目旨在从技术美学的视角,系统考察20世纪商业珠宝的审美机制。研究以鲍尔格曼的"设备范式"与"焦点实践"理论为主要分析框架,结合海德格尔的技术哲学思想,探讨工业铸造工艺革新如何将珠宝制作从手工艺美学转变为商业技术美学。

## 研究框架

### 核心理论
- **技术哲学基础**: 马克思技术哲学、海德格尔技术哲学、芒福德人文主义技术哲学
- **技术美学理论**: 从工艺美学到技术美学的转型、鲍尔格曼"设备范式"与"焦点实践"
- **设计美学**: 中国设计美学的演变、20世纪商业珠宝的技术美学特征

### 研究方法
- 谱系学方法: 梳理20世纪商业珠宝技术美学的历史演变节点
- 双轨分析框架: "技术驱动演变"与"美学形态变迁"的对应关系
- 经验层面分析: 运用鲍尔格曼理论解读不同类型商业珠宝的审美体验结构
- 三位一体评价: 功能—审美—文化的综合评价框架

## 主要研究内容

1. **技术哲学基础理论框架**
   - 技术哲学的学科边界与研究纲领
   - 人文主义技术哲学: 芒福德与批判传统
   - 现象学技术哲学: 从海德格尔到鲍尔格曼的经验转向
   - 当代技术哲学: 斯蒂格勒与AI时代的延伸

2. **技术美学的概念界定与历史发展**
   - 中国技术美学的理论奠基
   - 苏联技术美学传统及其影响
   - 中国技术美学的历史分期: 从工艺美学到技术美学

3. **中国设计美学的演变路径**
   - 中国设计美学的历史根基: 先秦器物美学
   - 20世纪中国现代设计的技术美学转型
   - 技术驱动的工业设计美学演变

4. **技术美学在具体领域的应用研究**
   - 传统工艺的技术美学分析
   - 建筑领域的技术美学谱系
   - 品牌视觉识别与商业美学
   - 数字技术与AI时代的技术美学新发展

## 研究创新点

- 填补技术美学在珠宝研究领域的空白
- 建立系统的20世纪商业珠宝技术美学分析框架
- 将抽象的技术哲学洞见落地于具体物质文化研究
